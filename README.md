### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/MaxDunkelx/Israel-cyber-academy.git
   cd Israel-cyber-academy
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

   ```bash

   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to see the application
